## 🎯 クイズ問題案

### 【問題1】インデント基礎 ⭐
**問題**: PEP8で推奨されているインデントのスペース数は？

A) 2スペース  
B) 4スペース  
C) 8スペース  
D) タブ文字

**正解**: B) 4スペース  
**解説**: PEP8では1つのインデントレベルにつき4スペースの使用を推奨しています。

---

### 【問題2】行の長さ制限 ⭐⭐
**問題**: PEP8で推奨されている1行の最大文字数は？

A) 72文字  
B) 79文字  
C) 88文字  
D) 100文字

**正解**: B) 79文字  
**解説**: コードは79文字、コメントやdocstringは72文字が推奨されています。

---

### 【問題3】命名規則クイズ ⭐⭐
**問題**: 以下のうち、PEP8に**最も適合する**変数名はどれ？

A) `userName`  
B) `user_name`  
C) `UserName`  
D) `user-name`

**正解**: B) `user_name`  
**解説**: 変数名にはsnake_case（小文字+アンダースコア）を使用します。

---

### 【問題4】クラス名の規則 ⭐⭐
**問題**: PEP8に従った正しいクラス名はどれ？

A) `http_server_error`  
B) `httpServerError`  
C) `HTTPServerError`  
D) `Http_Server_Error`

**正解**: C) `HTTPServerError`  
**解説**: クラス名はCapWords（PascalCase）を使用し、略語は全て大文字にします。

---

### 【問題5】コード修正問題 ⭐⭐⭐
**問題**: 以下のコードをPEP8に準拠するよう修正してください：

```python
def calculate_total(price,tax):
    result=price+price*tax
    return result
```

**修正例**:
```python
def calculate_total(price, tax):
    result = price + price * tax
    return result
```

**ポイント**: 
- カンマの後にスペース
- 演算子の前後にスペース

---

### 【問題6】インポート文の順序 ⭐⭐⭐
**問題**: PEP8に従った正しいインポート順序は？

```python
# A)
import sys
from mypackage import mymodule
import requests

# B)
import requests
import sys
from mypackage import mymodule

# C)
import sys
import requests
from mypackage import mymodule
```

**正解**: C)  
**解説**: 
1. 標準ライブラリ
2. サードパーティライブラリ
3. ローカルアプリケーション/ライブラリ

### 【問題7】コメントスタイル ⭐⭐
**問題**: 正しいインラインコメントはどれ？

A) `x = x + 1# インクリメント`  
B) `x = x + 1 # インクリメント`  
C) `x = x + 1  # インクリメント`  
D) `x = x + 1   #インクリメント`

**正解**: C) `x = x + 1  # インクリメント`  
**解説**: コードとコメントの間は最低2スペース、#の後に1スペース

---

### 【問題8】避けるべき変数名 ⭐⭐
**問題**: PEP8で使用を避けるべき単一文字変数名はどれ？

A) `l` (小文字のエル)  
B) `O` (大文字のオー)  
C) `I` (大文字のアイ)  
D) 上記すべて

**正解**: D) 上記すべて  
**解説**: これらは数字の1や0と区別しにくいため避けるべきです。

---

### 【問題9】空白行のルール ⭐⭐⭐
**問題**: トップレベルの関数定義の前後には何行の空白行を入れるべき？

A) 1行  
B) 2行  
C) 3行  
D) 空白行は不要

**正解**: B) 2行  
**解説**: トップレベルの関数とクラスの前後は2行、クラス内のメソッドは1行の空白行

---

### 【問題10】文字列クォート ⭐
**問題**: PEP8では単一引用符と二重引用符について何と述べている？

A) 単一引用符を推奨  
B) 二重引用符を推奨  
C) どちらでも良いが一貫性を保つ  
D) 文脈によって使い分ける

**正解**: C) どちらでも良いが一貫性を保つ  
**解説**: ただし、文字列内に引用符が含まれる場合は、エスケープを避けるため反対の引用符を使用

---

### 【問題11】関数引数のデフォルト値 ⭐⭐⭐
**問題**: PEP8に準拠した関数定義はどれ？

A) `def func(arg1, arg2 = None):`  
B) `def func(arg1, arg2=None):`  
C) `def func(arg1, arg2= None):`  
D) `def func(arg1, arg2 =None):`

**正解**: B) `def func(arg1, arg2=None):`  
**解説**: デフォルト値を示す=の前後にはスペースを入れません

---

### 【問題12】複合文の禁止 ⭐⭐
**問題**: PEP8で非推奨とされているコードはどれ？

A) `if x == 4: print(x)`  
B) `if x == 4:\n    print(x)`  
C) `x = 1; y = 2; z = 3`  
D) AとC両方

**正解**: D) AとC両方  
**解説**: 複合文（1行に複数の文）は一般的に非推奨です

---

### 【問題13】定数の命名 ⭐⭐
**問題**: 定数の正しい命名規則は？

A) `maxValue`  
B) `max_value`  
C) `MAX_VALUE`  
D) `MaxValue`

**正解**: C) `MAX_VALUE`  
**解説**: 定数は全て大文字でアンダースコア区切り

---

### 【問題14】演算子の優先度と空白 ⭐⭐⭐
**問題**: PEP8に最も適合するコードはどれ？

A) `y = x * 2 + 1`  
B) `y = x*2 + 1`  
C) `y = x * 2+1`  
D) `y = x*2+1`

**正解**: B) `y = x*2 + 1`  
**解説**: 優先度の低い演算子（+）の周りにスペースを入れ、高い演算子（*）は詰める

---

### 【問題15】コード穴埋め ⭐⭐⭐
**問題**: 以下のコードの__に入る適切な空白を答えてください：

```python
def__example(__arg1,__arg2__=__None__):
____return__arg1__+__arg2
```

**正解**:
```python
def example(arg1, arg2=None):
    return arg1 + arg2
```
