## 2026-07-18-01: 多オブジェクト機能は対象オブジェクトを明示する

- Quiz: 2026-07-20 / CREATE OR ALTER TABLEの宣言的DDL
- Source: user-feedback
- Phase: review
- Signal: CREATE OR ALTER を総称で出題したが、列リネーム等の論点は TABLE 固有。レビューで対象オブジェクトの明示を求められ修正した
- Cause: 機能ファミリー名で書いてしまい、問題が依拠するオブジェクト境界が曖昧になった
- Action: タイトル・問題文・選択肢・解説を CREATE OR ALTER TABLE に絞った。次回も多オブジェクト対応機能は、論点が依拠するオブジェクトを問題文で先に固定する
- Success metric: 同様の総称機能（CREATE OR ALTER / ALTER 系など）で、レビュー前に対象オブジェクトがタイトルまたは問題文に明示されている
