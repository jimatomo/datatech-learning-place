# dtsb-learning-place
web application of learning place for datatech

## 開発者向け情報

### 開発環境

- ローカル: Docker Desktop + Dev Container

### ローカルでの開発

```bash
# ローカルでの開発
npm run dev

# AWS SSO Config
aws configure sso

# SSO
aws sso login
```
